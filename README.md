# SAE-1.06-ControverseSociotechnique-WebSite

## Description : 
Réalisation d'un site web pour la SAE1.06 de première année. Le site devait être responsive grâce à l'utilisation des media queries. De plus, le site doit être simple d'utilisation, être composé de certains éléments obligatoires (nombre de pages, informations...). Ce site a été réalisé uniquement en HTML CSS par les élèves. L'utilisation de l'IA a été minimale, uniquement sur des questions au sujet du CSS pour des éléments que nous ne comprenions pas pourquoi ils n'interagissaient pas entre eux.

our réaliser le site web, nous avons réparti les tâches au sein des membres du groupe pour cette SAE 1.06.
